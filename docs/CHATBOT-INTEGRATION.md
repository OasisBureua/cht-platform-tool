# CHM Chatbot Integration

When ready to integrate the chatbot into CHT Platform.

## Embeddable widget
```html
<script src="https://chmbot.communityhealth.media/widget.js"></script>
```

## Inline embedding
```html
<div id="chm-chatbot" style="width:100%;height:600px;"></div>
<script src="https://chmbot.communityhealth.media/widget.js"
        data-mode="inline" data-container="chm-chatbot"></script>
```
